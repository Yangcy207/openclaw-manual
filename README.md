# OpenClaw 使用手册

本项目包含 OpenClaw 的完整使用文档，分为三个主要部分：

## 📁 目录结构

```
openclaw-manual/
├── windows-install/      # Windows 环境安装指南
├── cloud-deployment/     # 云端部署指南
└── skills-mcp-guide/     # Skill 和 MCP 工具使用指南
```

## 📚 文档说明

### [Windows 安装指南](./windows-install/README.md)
介绍如何在 Windows 系统上安装和配置 OpenClaw。

### [云端部署指南](./cloud-deployment/README.md)
介绍如何将 OpenClaw 部署到各种云平台（Docker、Fly.io、GCP 等）。

### [Skill 和 MCP 工具指南](./skills-mcp-guide/README.md)
介绍如何使用和添加 OpenClaw 的 Skill 和 MCP 工具。

## 🚀 快速开始

根据你的需求选择对应的文档：

- **本地开发**：查看 [Windows 安装指南](./windows-install/README.md)
- **生产部署**：查看 [云端部署指南](./cloud-deployment/README.md)
- **功能扩展**：查看 [Skill 和 MCP 工具指南](./skills-mcp-guide/README.md)
